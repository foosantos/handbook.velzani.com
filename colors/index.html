SKU Structure Using Primary & Secondary Colors

STYLE-SIZE-PRIMARYCOLOR-SECONDARYCOLOR

SNK01-42-BLK-WHT = Sneaker model 01, size 42, primarily black with white accents

BTL02-39-BRN-TAN = Boot model 02, size 39, primarily brown with tan accents

RUN03-41-RED-BLK = Running shoe model 03, size 41, primarily red with black accents

This structure eliminates the need for dedicated two-tone color codes, allowing for flexible color combinations using the existing color codes.
